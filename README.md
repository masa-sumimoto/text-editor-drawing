# Overview
ATOMエディタを使い16進数のカラーコードでお絵かきをしました。やってみた系です。
方法は以下の通りです。

## Getting Started
```
1.ATOMエディタをインストールする
2.カラーコード文字列の背景を着色してくれるパッケージをインストールする ※デモではpigmentsを使用。
3.テキストファイルを準備する ※デモでは.txtファイル
カラーコードを3行1セットとして書いてゆく
```

さあ皆様もLet's お絵かき！
